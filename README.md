Aşama 2,3 için sayfa adresi: http://127.0.0.1:5000/scoring

Aşama 4 için sayfa adresi: http://127.0.0.1:5000/indexing

Aşama 5 için sayfa adresi: http://127.0.0.1:5000/semantik
